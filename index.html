Introducción al Análisis Numérico

Al aplicar las Matemáticas a situaciones del mundo real, nos encontramos a menudo con problemas que no pueden ser resueltos analíticamente o de manera exacta. Estos problemas deben ser abordados con ayuda de procedimientos o métodos numéricos.

Un método numérico es un procedimiento mediante el cual se obtiene, casi siempre de manera aproximada, la solución de ciertos problemas realizando cálculos puramente aritméticos y lógicos.

La eficiencia en el cálculo de dicha aproximación depende, en parte, de las características especiales y limitaciones de los instrumentos de cálculo. Al emplear estos instrumentos se introducen distintos tipos de errores, y ese será el tema central de estudio.

Clasificación de los Errores

Errores Sistemáticos

Asociados a los instrumentos de medida, sometidos a influencias externas como cambios de temperatura, humedad, defectos de fabricación, impericia del experimentador, mala calibración, etc.

No pueden eliminarse totalmente, pero se minimizan mediante pruebas de control con objetos patrón.

Errores Inherentes

Surgen cuando aplicamos un modelo matemático que resulta ser una aproximación a la realidad.

Estos errores pueden ajustarse para lograr que sean mínimos, pero siempre estarán presentes en las simulaciones y modelos simplificados.

Errores por Redondeo

Se presentan cuando trabajamos con números en su expresión decimal, pero debemos prescindir de infinitas cifras.

Existen tres tipos de redondeo: por defecto, por exceso y al valor más próximo.

Errores de Truncamiento

Se presentan cuando interrumpimos un proceso infinito de cálculo después de un número finito de operaciones.

Ejemplo: aproximar una función mediante la suma de los n primeros términos de una serie infinita.

Fundamentos Teóricos

Error Absoluto

Diferencia entre el valor exacto (α) y el valor aproximado (a):

Δ(a) = |α − a|

Representa la magnitud del error sin considerar la escala del valor exacto. Tiene unidades de medida.

Error Relativo

Cociente entre el error absoluto y el valor exacto:

ε(a) = Δ(a) / |α| = |α − a| / |α|

Expresa la proporción del error en relación al valor exacto. No tiene unidades.

Cota Superior del Error Absoluto

Todo número real positivo Δ*(a) que verifica:

|α − a| ≤ Δ*(a)

Esto garantiza que el valor exacto se encuentra dentro del intervalo:

α ∈ [a − Δ*(a), a + Δ*(a)]

Cota Superior del Error Relativo

Todo número real positivo ε*(a) que verifica:

|ε(a)| ≤ ε*(a)

Se puede adoptar como cota superior:

ε*(a) = Δ*(a) / |a| (si a ≠ 0)

Ejemplo Práctico

Supongamos que aproximamos π (3.14159265...) con 3.14. Si establecemos una cota de error absoluto Δ* = 0.01, podemos garantizar que:

π ∈ [3.14 - 0.01, 3.14 + 0.01] = [3.13, 3.15]

El error absoluto sería Δ = |3.14159265 - 3.14| ≈ 0.00159

El error relativo sería ε = 0.00159/3.14159265 ≈ 0.000506 (0.0506%)

La cota de error relativo sería ε* = 0.01/3.14 ≈ 0.003185 (0.3185%)

Esto nos indica que tenemos una precisión de dos decimales correctos, y que el valor real de π se encuentra dentro del intervalo [3.13, 3.15].

¿Para qué sirven las cotas de error?

Garantía de precisión

Las cotas de error proporcionan una garantía sobre la precisión de una aproximación, permitiendo establecer límites dentro de los cuales se encuentra el valor exacto.

Comparación de métodos

Permiten comparar diferentes métodos numéricos o aproximaciones para determinar cuál proporciona mayor precisión con el mismo esfuerzo computacional.

Control de algoritmos

En algoritmos iterativos, las cotas de error se utilizan como criterio de parada, deteniendo las iteraciones cuando se alcanza la precisión deseada.

Análisis de estabilidad

Ayudan a evaluar la estabilidad de los métodos numéricos, determinando cómo se propagan los errores en los cálculos y si estos se mantienen dentro de límites aceptables.
